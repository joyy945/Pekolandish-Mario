# 兔田佩可拉瑪利歐小遊戲 (Usada Pekora Mario Demo)

這是一款使用 Python 與 Pygame 製作的 2D 橫向捲軸跳躍遊戲。
遊戲中你將扮演戴著共產帽的兔田佩可拉 (Usada Pekora)，挑戰精心設計的 10 個關卡！

## 遊戲特色

- **10 個獨特關卡**：從簡單的跳躍到稍具挑戰性的坑洞，保證每關皆可憑實力通關。
- **完美的像素風格**：包含可愛的佩可拉精靈圖、金幣、敵人與磚塊，並已完美處理透明去背。
- **順暢的操作體驗**：標準的瑪利歐物理引擎（重力、跳躍、碰撞偵測）。
- **進度自動儲存**：遊戲預設提供儲存與讀取功能，過關會自動存檔。
- **原生 720P 高解析度視窗**：提供舒適的 1280x720 遊戲畫面。

## 系統需求與運行方法

這個遊戲是使用 Python 撰寫的，因此需要您的電腦安裝有 Python 環境。

### 步驟 1：安裝 Python
如果您的電腦還沒有 Python，請先到 [Python 官方網站](https://www.python.org/downloads/) 下載並安裝。建議安裝 Python 3.8 或更新的版本。
*(注意：Windows 用戶在安裝時，請務必勾選「Add Python to PATH」選項)*

### 步驟 2：安裝必要模組 (Pygame)
打開終端機 (Terminal) 或 命令提示字元 (cmd / PowerShell)，並輸入以下指令來安裝遊戲所需的 Pygame 引擎庫：
```bash
pip install pygame
```

### 步驟 3：啟動遊戲
在終端機中，使用 `cd` 指令進入到解壓縮後的遊戲資料夾裡面 (擁有 `main.py` 的目錄)，然後輸入以下指令執行遊戲：
```bash
python main.py
```
*(如果 `python` 無法執行，在部分環境或 Mac 上可能需要輸入 `python3 main.py`)*

## 遊戲操作說明

在遊戲中，請使用您的鍵盤操作：

- **左右移動**：`左方向鍵 (←)` / `右方向鍵 (→)` 或 `A 鍵` / `D 鍵`
- **跳躍**：`上方向鍵 (↑)` 或 `W 鍵` 或 `空白鍵 (Space)`
- **暫停 / 存檔選單**：大寫或小寫 `P 鍵`
- **在暫停時儲存並回主選單**：大寫或小寫 `S 鍵`
- **強制返回主選單**：`ESC 鍵`
- **選單確認 / 死亡後重試**：`Enter 鍵`
